Tutor Delphia's Management (TDM) System

By Adam Levin.

Contributions by Justin Kantor. Consulting by Tristan Mills and Jared McGuire.

Project Overview

The TDM is used for managing tutors, tracking lessons, updating content (on www.tutordelphia.com) tracking leads and clients.

By using the TDM we were able to automate processes such as managing tutor payments, assigning clients to tutors and tracking lessons.

Toolset
  • PHP
  • MySQL
  • HTML, CSS and JavaScript
  • JQuery
  • Slim Framework
Lessons Learned

When working on toy projects and going through tutorials, it is hard to see that much of the challenge of programming comes from managing complexity. I learned many lessons regarding that, from the importance of carefully chosen variable and database names to design patterns to simply creating reusable code and templates.
