FAQ #01

How can I contact for support?

Please go to our profile page. Then mail us. We will reply you within 24 Hours.

Contact form active?

Yes, the contact form is active. First open the mail.php file. Then remove the default mail address & placed your mail address.

Are there any possibilities to edit default colors?

Sure, go to core-style.css and change color values to desire.

Can I use my own font in this template?

Sure, we used google fonts library. You can use your desire fonts.

FAQ #02

How do I will change Menu, Logo, Slider Text/Image and Footer Text?

Please see our documentation for edit thus.

Can I order to make a custom work for me?

Yes, it’s possible, we provide custom work only for our templates. Email us via the form at our profile page.

Would it possible to create more than one website within the regular license?

No, You can only use it on a website. Please read article for more information about regular license.

It can use any CMS (WordPress, Joomla & Others) Platform?

No. It’s just HTML template. If you want to use this template in CMS, please convert first.
